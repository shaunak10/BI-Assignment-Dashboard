# D2C Sales Analytics Dashboard

## Project Overview

This project analyzes Direct-to-Consumer (D2C) sales data using SQL, Python, and Business Intelligence techniques to derive actionable business insights. The project culminates in an executive dashboard developed using HTML, CSS, and JavaScript.

The objective is to evaluate sales performance, product performance, customer behavior, category trends, and channel effectiveness.

---

## Tech Stack

* Python
* Pandas
* SQL (SQLite)
* Jupyter Notebook
* HTML
* CSS
* JavaScript

---

## Project Components

### SQL Analysis

The following business questions were solved using SQL:

1. Customer Segment Analysis

   * Total Orders by Segment
   * Average MRP by Segment

2. Product Revenue Analysis

   * Revenue generated by each product

3. Channel Discount Analysis

   * Identification of channels offering the highest discounts

---

### Python Analysis

Created analytical dataframes to calculate:

* Category-wise Revenue
* Total Orders
* Average Order Value (AOV)

Revenue Formula:

Revenue = Quantity × MRP × (1 - Discount Percentage / 100)

---

### Dashboard Features

The executive dashboard includes:

* Total Revenue KPI
* Total Orders KPI
* Average Order Value KPI
* Top Revenue Product
* Highest Revenue Category
* Highest Discount Channel
* Category Performance Analysis
* Product Performance Analysis
* Customer Segment Analysis
* Business Insights Section

---

## Key Business Insights

* Wellness emerged as the highest revenue-generating category.
* Vitamin C Serum was the top-performing product.
* Gold customers generated the highest order volume.
* Amazon provided the highest average discount.
* Skincare recorded the highest Average Order Value.
